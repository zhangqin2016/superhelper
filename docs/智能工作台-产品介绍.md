# 智能工作台

**面向团队的 AI 桌面助手 · 产品简介**

版本 0.1 · 2026 年 5 月

---

## 这是什么？

**智能工作台**是一款安装在电脑上的桌面应用，帮你用自然语言完成日常工作：写方案、查资料、读网页、看图片、整理项目文件。

你可以把它理解成：

- 一个**始终在线的 AI 对话窗口**（像聊天软件一样好用）
- 一个**按文件夹划分的工作台**（不同项目互不干扰）
- 一个**开箱即用的工具箱**（联网搜索、打开网页、识图等能力已内置）

不需要在浏览器和多个网站之间来回切换，也不需要自己配置复杂的开发环境。

## 适合谁用？

| 人群 | 能做什么 |
|------|----------|
| 业务 / 运营 | 写文案、做调研、整理会议纪要 |
| 产品 / 设计 | 快速原型说明、竞品信息汇总 |
| 研发 / 测试 | 在指定项目目录下问代码、查文档、辅助排查 |

**不需要会写代码**，会打字、会选文件夹即可上手。

## 核心功能

### 1. 多工作空间（按文件夹管理）

- 左侧可以添加多个**工作空间**，每个对应电脑上的一个文件夹
- 不同客户的项目、不同部门的资料可以分开存放
- 切换工作空间时，对话记录和上下文互不串台

**举例：**「市场部资料」和「产品需求文档」各建一个工作空间，互不干扰。

### 2. 多对话（像微信多个聊天窗口）

- 每个工作空间里可以新建多条**对话**
- 每条对话有独立的历史记录
- 支持重命名、删除、归档
- 可以同时开多个对话，切换时界面会保留正在进行的回复进度

### 3. 流式回复与清晰展示

- 助手回答会**逐字显示**，不用干等一整段结果
- 支持 Markdown 排版（标题、列表、代码块等）
- 执行步骤会以卡片形式展示，方便了解「正在做什么」

### 4. 发送图片与附件

- 输入框支持**添加本地文件**
- 图片可由助手识别并描述内容（需联网）
- 适合「这张图是什么」「帮我看看截图里的错误」等场景

### 5. 联网能力（内置）

应用内置了联网工具，无需单独申请搜索 API：

| 能力 | 说明 |
|------|------|
| 网页搜索 | 查找新闻、资料、公开信息 |
| 打开网页 | 读取指定链接的正文并总结 |
| 图片识别 | 描述本地或网络图片内容 |

> 使用联网功能需要电脑能访问互联网。

### 6. 模型切换

- 设置面板中可切换**不同模型档位**（例如「偏质量」与「偏速度」）
- 网关与密钥由应用统一配置，普通用户无需填写技术参数
- 切换模型后，下一次发送消息时生效

### 7. 权限模式

- 可设置助手执行操作时的**权限级别**（如更谨慎 / 更自动）
- 适合对安全要求不同的团队环境

## 典型使用流程

```
安装应用 → 打开「智能工作台」
    ↓
点击「+」添加工作空间（选择项目文件夹）
    ↓
新建对话 → 输入问题或拖入图片
    ↓
查看流式回复 → 继续追问或切换其他对话
```

**第一次使用建议：**

1. 选一个你熟悉的项目文件夹作为工作空间  
2. 新建一条对话，用简单问题试一句，例如：「用三句话介绍这个文件夹大概是做什么的」  
3. 需要查最新资讯时，直接说：「帮我搜索……」

## 系统要求

| 项目 | 要求 |
|------|------|
| 操作系统 | **macOS**（Apple 芯片，推荐）· Windows 安装包需包含内置引擎 |
| 网络 | 对话与联网功能需要可访问外网（或公司允许访问的 API 网关） |
| 磁盘空间 | 建议预留 500 MB 以上（含应用与内置组件） |
| 其他 | **无需**用户自行安装 Node.js、npm 或命令行工具 |

### 平台说明（给 IT / 运维）

- **macOS（M 系列芯片）：** 安装包内已携带 AI 引擎，用户安装后可直接使用。  
- **macOS（Intel）：** 需确认安装包是否包含对应版本引擎。  
- **Windows：** 需在打包时放入 Windows 版引擎；当前开发版若未包含，需按发布清单补全后再分发。  

首次在 macOS 上打开未签名的安装包时，系统可能提示「无法验证开发者」，可在「系统设置 → 隐私与安全性」中选择仍要打开，或联系管理员完成企业签名。

## 数据存在哪里？

- **对话记录、工作空间列表：** 保存在本机用户目录下的应用数据文件夹中  
- **项目文件：** 始终在你选择的文件夹里，应用不会把整个硬盘扫一遍  
- **模型访问：** 通过应用内置的网关配置连接云端服务（具体地址由部署方设定）  

如需合规说明，请以贵司实际部署的网关、日志策略为准。

## 常见问题

**问：和网页版聊天工具有什么区别？**  
答：智能工作台绑定本地文件夹，适合长期跟进某个项目；多条对话、工作空间管理更适合日常办公节奏；部分能力（如固定项目目录下的协作）更贴近桌面场景。

**问：要不要会编程？**  
答：不需要。会输入中文问题、会选择文件夹即可。

**问：能否离线使用？**  
答：核心对话依赖云端模型，需网络。纯离线暂不支持。

**问：换电脑后记录还在吗？**  
答：对话记录存在本机。换机需自行迁移应用数据，或重新开始（以实际备份方案为准）。

**问：发消息没反应怎么办？**  
答：依次检查：① 网络是否正常；② 是否已添加工作空间并选中对话；③ 完全退出应用后重新打开；④ 联系管理员确认密钥与网关是否有效。
